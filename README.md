# Rota IA-Projeto
Sistema inteligente que organiza os pontos de paradas com eficiência e econômia
escolhendo a melhor rota para os determinados locais de embarque 
fazendo assim, um trajeto inteligente economizando a quilometragem e combustivel
e o princpal, o tempo de duração da viagem. \n
